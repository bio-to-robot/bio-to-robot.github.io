# Bio-to-Robot Transfer Project Website

Static GitHub Pages site for the paper:

**Bio-to-Robot Transfer of Fish Sensorimotor Dynamics via Interpretable Model**

## Files

- `index.html` — single-page project website
- `biorob.pdf` — paper PDF linked from the page
- `teaser-fish-robot.png`, `signal-pipeline.png`, `arx-workflow.png`, `robot-results.png`, `paper-cover.png` — figure assets cropped from the paper for the project page
- `pku-logo.png`, `ocean-research-logo.jpg` — affiliation logos from the original template

## Deploy on GitHub Pages

1. Create a new repository named something like `bio-to-robot-fish.github.io` or any project repo.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Select **Deploy from branch**, choose `main`, and use `/root`.
5. Save. GitHub Pages will publish the site after the first build.

## Customization

Search in `index.html` for these placeholders and update as needed:

- Code/Data links in the `#code` section
- Video or supplementary-material buttons if you have public URLs
- `og:url` if you want social-preview metadata for a final public URL
